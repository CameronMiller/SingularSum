# Singular Sum

A static website for [singularsum.com](https://singularsum.com/), inspired by Stripe's clean design.

## Tech Stack

- **HTML** – Static pages
- **Tailwind CSS** – Via CDN (no build step)
- **Inter** – Google Fonts

## Structure

- `index.html` – Home / landing page
- `technology.html` – Patented technology overview
- `contact.html` – Contact info and inventors
- `favicon.svg` – Site icon

## Preview Locally

```bash
npx serve .
```

Then open http://localhost:3000 (or the port shown).

## Deployment

Upload the HTML files and `favicon.svg` to any static host. No build step required. Works with Netlify, Vercel, GitHub Pages, S3, or any web server.
